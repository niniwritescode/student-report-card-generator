# Student Report Card Generator

A simple Flask web application that generates student report cards and allows downloading them as a PDF.

## Features
- Enter student details
- Calculate total marks and percentage
- Automatic grade generation
- Downloadable PDF report card

## Tech Stack
- Python
- Flask
- HTML/CSS
- ReportLab (PDF generation)

## How to Run

1. Install dependencies
pip install flask reportlab

2. Run the application
python app.py

3. Open browser
http://127.0.0.1:5000
